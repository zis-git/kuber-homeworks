
# Домашнее задание к занятию «Установка Kubernetes»

## Задача

Необходимо развернуть Kubernetes‑кластер с использованием **kubeadm** и проверить его работоспособность.

---

# Подготовленная инфраструктура

Кластер развернут на виртуальной машине со следующими параметрами:

```
IP адрес: 192.168.88.32
ОС: Ubuntu 24.04.1 LTS
Kernel: 6.8.0-101-generic
```

Используемые компоненты Kubernetes:

```
Kubernetes: v1.30.14
Container Runtime: containerd 1.7.28
CNI: Flannel
```

Кластер был инициализирован командой:

```bash
kubeadm init
```

После инициализации был установлен сетевой плагин **Flannel**.
